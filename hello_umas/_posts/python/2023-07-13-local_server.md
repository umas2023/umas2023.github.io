- 假如vue项目已经打包好了dist文件夹，可以用这个命令来启动一个web服务器

{% raw %}
```bash
python.exe -m http.server 8000
python.exe -m http.server 8000 --directory D:\s-linux\project\localviewer\htmlviewer\data
```
