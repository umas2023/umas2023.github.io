- 深度监听，监听对象的属性
```
watch(
    getFlagCloud,
    () => {
    setSkeletonFlag();
    },
    { deep: true }
);
```

- 如果监听对象是props.data，会报错：A watch source can only be a getter/effect function, a ref, a reactive object, or an array of these types. 
    ○ 使用箭头函数：
    
    watch(()=>props.data, () => {
        console.log(props.data)
    }, { deep: true });
