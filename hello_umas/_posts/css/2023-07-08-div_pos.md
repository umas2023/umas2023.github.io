## 上下左右居中

- div自身靠左/靠右

{% raw %}
```html
<div class="left"></div>

<style lang="scss" scoped>
.left {
        float: left;
    }
</style>
```
{% endraw %}

- div内部元素水平居中

{% raw %}
```html
<div class="body">
    <div class="inner"></div>
</div>

<style lang="scss" scoped>
.body {
        text-align: center;
    }
</style>
```
{% endraw %}


- div内部元素垂直居中

{% raw %}
```html
<div class="layout-header">
    <div class="inner"></div>
</div>

<style lang="scss" scoped>
div.layout-header {
        display: flex;
        align-items: center;
    }
</style>
```
{% endraw %}
