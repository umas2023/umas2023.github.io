## 基本语法

- 文字不换行

{% raw %}
```
white-space: nowrap
```
{% endraw %}


- 文字对齐

{% raw %}
```
text-align: right; /* 文字靠右对齐 */
```
{% endraw %}


- 禁止文本选中
{% raw %}
```css
user-select: none; // 禁止文本选中
```
{% endraw %}
