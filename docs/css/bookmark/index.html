css效果库

1. 鬼畜震动

  • CSShake
  • npm i csshake -s
  • 使用时候在style里import
<div class="shake shake-crazy">
<style>
@import '~csshake';
div.shake-box {
    display: inline-block;
    margin: auto;
    width: 300px;
    height: 200px;
    background-color: red;
}

2. 出入动效

  • Animate.css
  • npm install animate.css –save
<div class="animate-box animate__animated animate__bounce">
<style lang="scss" scoped>
@import 'animate.css';
