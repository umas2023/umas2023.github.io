linux: iperf3网络测速命令

服务器和客户端同时启动命令，测试带宽

  • 安装
  • 服务器启动
  • 客户端测试带宽

安装

apt install iperf3

服务器启动

# 默认端口号5201
iperf3 -s
# -p指定端口
iperf3 -s -p 8888

客户端测试带宽

  • 基本命令
# -c指定地址开始测试
iperf3 -c 192.168.1.1
# -P设置多个并行数据流
iperf3 -c 192.168.1.1 -P 3
# -R反向测试
iperf3 -c 192.168.1.1 -R
