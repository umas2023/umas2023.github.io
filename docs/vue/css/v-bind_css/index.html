vue: v-bind css 通过v-bind在style中使用变量

通过v-bind在style中使用变量

  • style中引用script变量：
<script setup>
const theme = {
    color: "red",
};
</script>
    
<template>
    <p>hello</p>
</template>
    
<style scoped>
p {
    color: v-bind("theme.color");
}
</style>
  • 加上”px”后缀，除了computed添加之外，还可以：
<style scoped>
p {
 font-size: calc(1px * v-bind(size));
}
</style>
  • 也可以设置多个style组
<template>
 <p :class="classes1.red">This should be red</p>
</template>

<style module="classes1">
.red {
 color: red;
}
</style>

<style module="classes2">
.red {
 color: green;
}
</style>
