leetcode: 斐波那契数列

动态规划的思想也能解这一题

  • 题目
  • 题解

题目

  • 链接：https://leetcode.cn/problems/fibonacci-number/description/?envType=study-plan-v2&envId=dynamic-programming

题解

  • 题目就是给n求fib(n)
  • 可以用动态规划的方法来理解
  • 边界条件是dp(0) = 0, dp(1) = 1

  • 状态转移方程是 dp(n) = dp(n-1)+dp(n-2)

  • 为了节省空间，可以用p、q、r仅保留dp的后三位，滚动赋值
class Solution {
    public int fib(int n) {
        if (n < 2) {
            return n;
        }
        int p = 0, q = 0, r = 1;
        for (int i = 2; i <= n; ++i) {
            p = q; 
            q = r; 
            r = p + q;
        }
        return r;
    }
}
